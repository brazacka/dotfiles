My dotfiles
===========

My application configuration files, obvs.

Usage
-----

Clone or fork this project, then just symlink these bad boys into your home directory:

```bash
$ ln -s .tmux.conf ~
$ ln -s .vimrc ~
$ ln -s .zshrc ~
```
